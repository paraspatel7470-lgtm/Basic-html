Lecture 1: Introduction To Web Development

How Does Internet Work

Principle 1: The Need to Share Information

  • Information is on one computer and needs to be on another, possibly far away.
  • The simplest solution is physically carrying data (USB), but it's slow and unscalable.
  • A cable can connect two computers directly.
  • Data can be sent using electrical or light signals representing bits (1s and 0s).
  • This is the most basic network — a direct link between two computers.

Principle 2: The Problem of Scale

  • Directly connecting every computer to every other is impractical as the number grows.
  • A central device, called a Switch, connects all computers in a local area.
  • The switch forwards messages only to the intended recipient.
  • This creates a Local Network, efficient for homes and offices.

Principle 3: The Problem of Connecting Networks

  • Principle 3: The Problem of Connecting Networks
  • It's impractical to run wires between all computers in all networks.
  • A Router connects two or more networks and routes messages between them.
  • By connecting routers, we build a global network.

Principle 4: The Problem of Finding Anyone (Addressing)

  • Millions of devices need unique addresses to communicate.
  • Like postal addresses, devices use IP Addresses to identify themselves.
  • Routers use IP addresses to decide where to send packets next.
  • Routers dont need the full path; they only need to know the next step.

Principle 5: The Problem of Reliability and Fairness

  • Sending large files at once monopolizes the network and is unreliable.
  • Files are broken into small, numbered Packets.
  • Each packet contains:
    • Data payload
    • Sequence number for reassembly
    • Sender’s IP address
    • Destination IP address
  • Packets can travel different routes and be reassembled at the destination.

IP Address

An IPv4 (Internet Protocol version 4) address is the classic IP address format everyone is used to seeing.

Public IP Address and Private IP Address

* A Public IP address is your global, unique address on the internet.

*A Private IP address is a local address used only within your own private network (e.g., your home Wi-Fi, your office network).

DNS(DOMAIN NAME SYSTEM)

DNS is the global, distributed system that translates the human-friendly domain names into the computer-friendly IP addresses. Without it, you'd have to memorize hundreds of IP addresses to browse the web.
