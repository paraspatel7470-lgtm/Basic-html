Lecture 2: HTML : Hyper Text Markup Language

What is HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and structure the content of a webpage.

  • HyperText: This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.
  • Markup Language: This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.


Tags:

Sno. Tags in HTML Uses Of Tags
1. heading tag (h1 tag to h6)  Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.
2. The Paragraph (P) The paragraph tag is the most common tag you'll use. It's for grouping sentences and blocks of text together.
2. The Line Break Tag (br) This tag tells the browser, "Stop writing on this line and immediately start on the next one." It's like hitting the Enter key once in a poem or an address.
3. The Anchor Tag (a) The Anchor tag is what makes the web "hypertext." It is used to create a hyperlink to another webpage, a file, or a location within the same page.
4. The Image Tag (img) The Image tag is a self-closing tag used to embed an image onto your page.

Attributes And Elements:

Sno. Attributes and Elements Uses of attributes and elements
1. HTML Links - The target Attribute By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.
2. List Elements((ul), (ol), (li)) These tags are used to create lists. This is a perfect example of nesting.
3. HTML Horizontal Rules (hr) The (hr) tag defines a thematic break in an HTML page, and is most often displayed as a horizontal rule.
