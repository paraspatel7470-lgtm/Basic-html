Lecture 4: File Path System

Relative File Paths

A relative file path gives directions to a file starting from the location of the file you are currently in. You will use this 99% of the time for linking your own files together (images, CSS, other HTML pages).

Absolute File Paths

An absolute file path gives the full, complete URL to a resource on the web. It starts with http:// or https://.

When to use it

  • Linking to another website.
  • Using a font from Google Fonts.
  • Using an image from an image-hosting service.
Sno. Element and Attribute Uses
1. div A (div) is a container element in HTML that is used to group together other elements. It stands for division or section of a page.
2. class The class attribute in HTML is used to assign a name (or multiple names) to an element, which helps you group elements together for styling with CSS or to manipulate them with JavaScript.
3. Id The id attribute is used to give a unique identifier to an HTML element. It helps you target a specific element when styling with CSS or manipulating with JavaScript.
