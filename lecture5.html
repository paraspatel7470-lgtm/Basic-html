What is an HTML form ?

An HTML form is a section on a webpage where users can enter data and send it to a server. Forms are used to collect information such as names, emails, passwords, feedback, and more.

Common Input Types

Sno. Types Uses
1. text For short text input
2. Email For email addresses
3. Password For entering passwords
4. Radio For selecting one option
5. checkbox For selecting multiple options
6. submit To submit the form
7. date For selecting a date
